=======================================
 Справка "Менеджер академии студентов"
=======================================

Базовые команды, использующиеся в сочетании с другими командами (см. ниже):
  ls|list <команда> -- отобразить список объектов
  add <команда> <аргументы> -- добавить объект
  assign|assigntask <команда> <аргументы> -- назначить задачу исполнителю
  eval <команда> <аргументы>, 
  evaluate <команда> <аргументы> -- выставить оценку за задачу
  prior <команда> <аргументы> -- создать приоретизрованных списков менторов и студентов
  import <формат> <путь к файлу> -- импорт объектов из файла
  export <формат> <путь к файлу> -- экспорт объектов в файл

Отображение списка объектов:
  ls ment|mentor -- вывести список менторов	
  ls stud|student -- вывести список студентов	
  ls task -- вывести список задач
  ls team -- вывести список команд и студентов в них
  ls taskstud -- вывести список назначенных задач студентам
  ls taskteam -- вывести список назначенных задач командам
  ls priorstud|priorstudent -- вывести приоретизированный список менторов у студентов	
  ls priorment|priormentor -- вывести приоретизированный список студентов у менторов

Добавление объектов:
  add ment|mentors <имя> -- добавление менторов
  add stud|student <имя> -- добавление студента
  add team <имя> -- добавление команды
  add task <имя> -- добавление задачи

Назначение задач исполнителям:
  assign stud <Id задачи> <Id студента>, 
  assign student <Id задачи> <Id студента> -- назначить задачу студенту 
  assign team <Id задачи> <Id команды> -- назначить задачу команде

Выставление оценки:
  eval stud <Id задачи> <Id студента> <Оценка>, 
  eval student <Id задачи> <Id студента> <Оценка> -- выставить оценку за задание студенту
  eval team <Id задачи> <Id команды> <Оценка> -- выставить оценку за задание команде

Создание приоретизрованных списков менторов и студентов:
  prior ment|mentors <Id ментора> <Список студентов> -- установить приоретизированный список студентов для ментора
  prior stud|student <Id студента> <Список менторов> -- установить приоретизрованный список менторов для студента
    где <Список студентов> и <Список менторов> -- приоретизированный список в кавычках,
        в котором элементы расположены через запятую в порядке следования приоритетов
  Например: prior stud 7 "3, 4, 2, 1"

Распределения студентов среди менторов в соответствии с приоритизированными списками:
  formgroups

Импорт:
  import json <имя файла> -- импорт объектов из JSON

Экспорт:
  export json <имя файла> -- экспорт объектов в JSON

Другие команды:
  filltest -- добавить тестовые данные
  help -- вывод справки

Примечание:
  Знаком '|' разделяются альтернативные написания одной и той же команды. 
  Необходимо вводить только один вариант команды одновременно.
